# Todo List with Vue.js and Vuex
利用 Vue.js 與 Vuex 簡單實作一個 Todo List。

## 功能介紹
實作的功能有

- 新增一個 todo
- 修改一個 todo的內容
- 變更一個 todo 的狀態：已完成或未完成
- 刪除一個 todo
- 切換檢視模式：看全部、已完成或未完成的 todo
- 在檢視按鈕上顯示目前全部、已完成和未完成的 todo 數量
